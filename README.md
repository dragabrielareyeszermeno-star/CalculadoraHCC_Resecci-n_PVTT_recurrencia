# Algoritmo de decisión quirúrgica en HCC (resección / PVTT / recurrencia)

## Autora
Dra. Miriam Gabriela Reyes Zermeño

## Descripción
Herramienta clínica interactiva para la toma de decisiones en carcinoma hepatocelular (HCC), integrando variables de función hepática, carga tumoral, biología tumoral y contexto terapéutico.

## Objetivo
Apoyar la toma de decisiones en:
- Resección hepática
- Trasplante hepático
- Downstaging
- Manejo de recurrencia
- HCC con invasión portal (PVTT)

## Base científica
Desarrollado a partir de:
- Malik AK, Geh D, Evans TRJ, et al. *Nat Rev Gastroenterol Hepatol*. 2025
- Guías BCLC, EASL, AASLD
- Evidencia multicéntrica contemporánea

## Uso
Herramienta educativa y de apoyo clínico.  
No sustituye juicio clínico especializado.
